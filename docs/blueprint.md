# **App Name**: FinanceFlow

## Core Features:

- Expense Tracking: Log all monthly expenses with categorization (e.g., groceries, rent, entertainment).
- Income Recording: Record all income sources, both fixed and variable, for a clear financial picture, according to the given spreadsheet example.
- Monthly Budgeting: Set monthly budget targets for different spending categories.
- AI-Powered Budget Insights: Use an AI tool to analyze spending patterns and suggest optimizations. The LLM determines when a specific insight should be suggested to the user, for example warning the user when their spending in a category exceeds 80% of the budgeted amount.
- Monthly and Annual Projections: Generate projections for monthly and annual expenses/income based on historical data.
- Customizable Categories: Allow users to create and manage custom spending categories.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey trust and stability in financial management.
- Background color: Very light purple (#F3E5F5) to provide a calm and clean backdrop.
- Accent color: Blue-Violet (#3F51B5) for interactive elements, providing visual interest.
- Body and headline font: 'Inter' for a modern and easily readable experience.
- Use minimalistic icons to represent each expense category, making it visually intuitive.
- Design a clean and intuitive layout that presents key financial data in a clear format, utilizing graphs and charts effectively.
- Incorporate subtle animations and transitions to enhance user engagement and provide feedback on actions.
